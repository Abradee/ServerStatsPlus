ServerStatsPlus Docs

These are the documentations and wiki for the Minecraft plugin called ServerStatsPlus.

NOTE:

This is a 100% unfinished project and none of this information is permenant.

About

This is a plugin that allows you to see detailed stats and data about your Minecraft server. This data can be viewed from a web browser and shows an extensive list including:

  1. Server TPS
  2. Server RAM Usage
  3. Server CPU Usage
  4. Server Chunk Stats
  5. Server Players
  6. Player Session Duration
  7. Player Ping
  8. Player Money
  9. Player LuckPerms Permissions
  10. Player LuckPerms Prefixes/Suffixes
  11. Punishment Tracking
  12. Chat Logs
  13. Death Logs
  14. Plugin Lists
